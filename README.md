# Aplikasi Sistem Akademik

Aplikasi Sistem Akademik adalah platform berbasis Android yang dirancang untuk mengelola data dan proses administrasi akademik kampus secara terintegrasi. Proyek ini dikembangkan menggunakan Android Studio dengan sistem build berbasis Gradle.

## Fitur Utama
* **Manajemen Data Akademik:** Pengelolaan data mahasiswa, dosen, serta informasi kelas.
* **Kartu Hasil Studi (KHS):** Input, rekapitulasi, dan monitoring nilai akademik mahasiswa.
* **Penjadwalan Kuliah:** Pengaturan ruang, waktu, dan mata kuliah secara sistematis.
* **Presensi Perkuliahan:** Pencatatan kehadiran digital untuk dosen dan mahasiswa.

## Prasyarat dan Teknologi
* Android Studio (versi terbaru disarankan)
* Java / Kotlin
* Gradle Build Tool

## Struktur Direktori
* `.idea/` : Konfigurasi lingkungan kerja Android Studio.
* `app/` : Modul utama aplikasi yang berisi kode sumber, aset, dan tata letak UI.
* `gradle/` : Konfigurasi dan wrapper untuk otomatisasi build proyek.
